## Quick Start

- Type `/` anywhere to open the **slash command menu**
- Press `Cmd+F` (or `Ctrl+F`) to **find and replace** in the current document
- Press `Cmd+P` to **quick open** any file
- Click the ==highlighted text== to see **highlight marks**
- Use `[[wiki links]]` to connect notes — try clicking [[feature-showcase]]
- Pin files with the star icon in the file tree

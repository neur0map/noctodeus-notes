# Markdown Cheatsheet

Quick reference for all supported markdown syntax.

## Headings

# Heading 1

## Heading 2

### Heading 3

## Emphasis

| Syntax | Result |
| --- | --- |
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strike~~` | ~~strike~~ |
| `code` | `code` |
| `==highlight==` | ==highlight== |
| `H~2~O` | H~2~O |
| `x^2^` | x^2^ |

## Links

- Standard: `[text](url)`
- Wiki: `[[note-name]]`
- Auto-linked URLs: [https://noctodeus.app](https://noctodeus.app)

## Images

```
![alt text](path/to/image.png)
![alt text](path/to/image.png =300x)
```

## Lists

```
- Bullet item
1. Numbered item
- [ ] Task (unchecked)
- [x] Task (checked)
```

## Code

Inline: surround with backticks.

Block: triple backticks with optional language.

## Tables

```
| Column | Column |
| --- | --- |
| Cell | Cell |
```

## Other

- `---` for horizontal rule
- `> text` for blockquote
- `[[note]]` for wiki link
- `/` for slash commands

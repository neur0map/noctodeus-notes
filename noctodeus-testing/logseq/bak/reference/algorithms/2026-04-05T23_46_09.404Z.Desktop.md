# Algorithm Notes

Quick reference for common algorithms with math notation and code.

## Big-O Complexity

| Algorithm | Time | Space |
| --- | --- | --- |
| Binary Search | $O(\log n)$ | $O(1)$ |

| Dijkstra | $O(E \log V)$ | $O(V)$ |

## Binary Search

The key idea: repeatedly halve the search space. Time complexity is $O(\log n)$.

```python
def binary_search(arr, target):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
```

## Math Formulas

### Inline Math

Euler's identity: $e^{i\pi} + 1 = 0$

The golden ratio: $\varphi = \frac{1 + \sqrt{5}}{2} \approx 1.618$

Pythagorean theorem: $a^2 + b^2 = c^2$

### Block Math

The quadratic formula:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Summation of first $n$ natural numbers:

$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$

Bayes' theorem:

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

Normal distribution:

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}$$

Matrix multiplication:

$$C_{ij} = \sum_{k=1}^{n} A_{ik} \cdot B_{kj}$$

### Calculus

Derivative definition: $f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

Fundamental theorem of calculus:

$$\int_a^b f(x)\,dx = F(b) - F(a)$$

## Sorting — Merge Sort

Divides array in half, sorts each, merges. Stable sort with $O(n \log n)$ guaranteed.

```javascript
function mergeSort(arr) {
  if (arr.length <= 1) return arr;
  const mid = Math.floor(arr.length / 2);
  const left = mergeSort(arr.slice(0, mid));
  const right = mergeSort(arr.slice(mid));
  return merge(left, right);
}

function merge(a, b) {
  const result = [];
  let i = 0, j = 0;
  while (i < a.length && j < b.length) {
    result.push(a[i] <= b[j] ? a[i++] : b[j++]);
  }
  return result.concat(a.slice(i), b.slice(j));
}
```

## Graph — Dijkstra's Algorithm

Shortest path from source to all vertices. Uses a priority queue. Complexity: $O(E \log V)$.

```rust
fn dijkstra(graph: &Vec<Vec<(usize, u64)>>, start: usize) -> Vec<u64> {
    let n = graph.len();
    let mut dist = vec![u64::MAX; n];
    let mut heap = BinaryHeap::new();
    dist[start] = 0;
    heap.push(Reverse((0, start)));
    while let Some(Reverse((cost, u))) = heap.pop() {
        if cost > dist[u] { continue; }
        for &(v, w) in &graph[u] {
            if dist[u] + w < dist[v] {
                dist[v] = dist[u] + w;
                heap.push(Reverse((dist[v], v)));
            }
        }
    }
    dist
}
```

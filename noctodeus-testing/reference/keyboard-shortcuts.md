# Keyboard Shortcuts

All available shortcuts. Cmd on macOS, Ctrl on Windows/Linux.

## Navigation

| Shortcut | Action |
| --- | --- |
| Cmd+P | Quick open file |
| Cmd+Shift+P | Command palette |
| Cmd+B | Toggle sidebar |
| Cmd+\ | Toggle right panel |
| Cmd+Shift+B | Collapse sidebar |

## Editor

| Shortcut | Action |
| --- | --- |
| Cmd+F | Find in document |
| Cmd+H | Find and replace |
| Cmd+B | Bold |
| Cmd+I | Italic |
| Cmd+U | Underline |
| Cmd+Shift+X | Strikethrough |
| Cmd+E | Inline code |
| Cmd+Shift+H | Highlight |
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |

## Files

| Shortcut | Action |
| --- | --- |
| Cmd+N | New note |
| Cmd+Backspace | Delete file |
| / | Slash command menu |
| [[ | Wiki link suggest |
